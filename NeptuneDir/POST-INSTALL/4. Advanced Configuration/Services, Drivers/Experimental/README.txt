BareBones Batch will configure the most minimal service/driver configuration for windows.

THIS IS NOT RECOMMENDED FOR LAPTOPS OR AMD SYSTEMS. YOU MAY BSOD.

If you want to test this, please make a restore point first, or if you are experienced enough, you can recover the OS through loading the SYSTEM hive in the windows setup.

If you experience any problems booting into the OS, please report it, and it will be mitigated.
